# Api de filmes 

Projeto pessoal criado para estudo e prática, com base no curso DevQuest.

## Requisitos

- Listar os filmes disponíveis
- Cadastrar um filme
- Atualizar dados de um filme
- Remover um filme
- Filtrar filmes por um gênero (ex: Drama)

## Tecnologias Utilizadas

- NodeJS
- PostgreeSQL
